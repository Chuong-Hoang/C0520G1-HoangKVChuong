☆ Bài tập 1: switch-case and if else if

  let browser = prompt("Enter your browser name:");
    switch (browser) {
        case 'Edge':
            alert("You've got the Edge!");
            break;
        case 'Chrome':
        case 'Firefox':
        case 'Safari':
        case 'Opera':
            alert("Okey, we support these browsers too");
            break;
        default:
            alert("We hope that this page looks ok!");
    }

☆ Bài tập 2:  Rewrite if-programme with switch-case

  let a = parseInt(prompt('a?', ''));
    switch (a) {
        case 0:
            alert(0);
            break;
        case 1:
            alert(1);
            break;
        case 2:
        case 3:
            alert('2,3');
            break;
    }

☆ Bài tập 3: Using if-else structure

    let value = parseInt(prompt("Enter a number:"));
    if (value > 0) {
        alert(1);
    } else if (value < 0) {
        alert(-1);
    } else {
        alert(0);
    }
