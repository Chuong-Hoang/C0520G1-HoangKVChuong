Tóm tắt nội dung Tổng Quan JavaScript

  • JavaScript là ngôn ngữ lập trình chạy trên các trình duyệt
  • JavaScript được sử dụng rộng rãi trên các website
  • Các cách nhúng JavaScript vào trang HTML
    1. Viết trực tiếp trên thẻ html (hạn chế dùng)
    2. Viết trong thẻ <script> và sử dựng thuộc tính src
    3. Viết trong file .js và nhúng vào html sử dụng thẻ <script>
    4. Sử dụng file .js online nhúng vào html bằng thẻ <script>
  • JavaScript thực thi từ trên xuống dưới nên để web tải nhanh thì chúng ta nên chèn mã JavaScript ở cuối trang HTML
  • Input và Output đối với JavaScript
      Input:
    • Dùng lệnh prompt()
    • Dùng thẻ input trong thẻ <form>
      Output:
    • Sử dụng lệnh alert()
    • Sử dụng lệnh console.log()
    • Sử dụng lệnh document.write()
    • Sử dụng lệnh document.getElementById().innerHTML
    • Sử dụng lệnh document.getElementById().innerText
